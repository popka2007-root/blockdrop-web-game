# БлокДроп

Браузерная игра с падающими блоками на русском языке.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npm start
```

После запуска сервер будет доступен на:

```text
http://localhost:8787
```

## Онлайн-комнаты

Онлайн-комнаты работают через `server.js`.

Локально:

```bash
npm start
```

В игре откройте `Онлайн-комната` и укажите:

```text
ws://localhost:8787
```

Для друзей сервер нужно развернуть как Node.js приложение на Render, Railway или VPS. GitHub Pages подходит для одиночной игры и челлендж-кодов, но не запускает Node.js сервер.

## Возможности

- 4 режима игры: Классика, 40 линий, Дзен, Хаос.
- Выбор сложности.
- Призрачная фигура.
- Запас фигуры.
- Три следующие фигуры.
- Челлендж-код для одинаковой последовательности фигур.
- Онлайн-комнаты с результатами друзей в реальном времени.
- Статистика, достижения, рекорды.
- PWA manifest и service worker.
