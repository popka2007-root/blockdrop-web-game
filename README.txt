Тетрис
========

Как играть без установки:
1. Открой файл index.html в браузере.
2. Нажми "Начать игру".
3. Управляй кнопками, свайпами или клавиатурой.

Управление:
- Влево/вправо: стрелки, кнопки или свайп.
- Поворот: кнопка "Поворот", стрелка вверх, W или тап по полю.
- Быстрый сброс: пробел, Z, свайп вниз или кнопка "Сброс".
- Запас фигуры: C или кнопка "Запас".
- Пауза: P, Esc или кнопка паузы.

Онлайн с друзьями:
1. Нужен Node.js сервер.
2. В этой папке запусти npm start.
3. Открой http://localhost:8787.
4. В игре открой "Онлайн-комната".
5. Для игры через интернет размести server.js на Render, Railway или VPS.

GitHub Pages подходит для одиночной игры. Для настоящего PvP нужен сервер, потому что атаки и комнаты работают через WebSocket.
