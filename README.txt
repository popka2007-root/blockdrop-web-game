БлокДроп Веб

Быстрый запуск на Windows:
1. Распакуй папку.
2. Дважды нажми Запустить_игру.bat
3. Игра откроется в браузере.

Быстрый запуск на iPhone:
1. Отправь index.html на iPhone.
2. Открой файл через приложение Файлы.
3. Если понадобится, нажми Поделиться -> Открыть в Safari.

Установка как приложение:
1. Выложи папку BlockDrop_Web на любой HTTPS-хостинг.
2. Открой ссылку в Safari на iPhone.
3. Нажми Поделиться -> На экран Домой.

Режимы:
- Классика: обычная игра на рекорд.
- 40 линий: цель очистить 40 линий.
- Дзен: спокойная скорость без давления.
- Хаос: периодически появляются линии снизу.

Управление:
- Влево / Вправо двигают фигуру.
- Поворот вращает фигуру.
- Вниз ускоряет падение.
- Сброс мгновенно отправляет фигуру вниз.
- Запас сохраняет текущую фигуру.
- Свайп влево или вправо двигает фигуру.
- Свайп вверх или тап поворачивает фигуру.
- Свайп вниз делает резкий сброс.

Горячие клавиши:
- A / D или стрелки: движение.
- W / Up / X: поворот.
- S / Down: мягкое падение.
- Space / Z: резкий сброс.
- C: запас фигуры.
- P / Esc: пауза.

Возможности:
- 4 режима игры.
- Выбор сложности.
- Призрачная фигура.
- Запас фигуры.
- Три следующие фигуры.
- Комбо и бонусы.
- Цели и прогресс.
- Темп и ранг игрока.
- Миссии в партии.
- Статистика и таблица лучших игр.
- Достижения.
- Поделиться результатом.
- Код челленджа для одинаковой последовательности фигур у друзей.
- Темы оформления.
- Контрастная палитра.
- Настройки сетки, опасной зоны, частиц, вибрации и громкости.
- Автосохранение партии.
- PWA-файлы для установки на экран телефона через хостинг.

Как играть с друзьями:
1. Открой игру.
2. Нажми Новый код в меню.
3. Нажми Поделиться.
4. Отправь другу код или ссылку.
5. Друг открывает игру с тем же кодом, режимом и сложностью.
6. У вас будет одинаковая последовательность фигур, можно честно сравнить результат.

Онлайн-комнаты:
1. Установи Node.js.
2. Открой терминал в папке BlockDrop_Web.
3. Запусти npm start
4. Открой http://localhost:8787
5. В игре нажми Онлайн-комната.
6. Сервер для локального теста: ws://localhost:8787
7. Введи комнату и имя, нажми Подключиться.
8. Для турнира выбери 3-8 игроков, таймер и нажми Старт турнира.

Чтобы друзья играли онлайн:
1. Выложи папку BlockDrop_Web на Render, Railway или VPS как Node.js приложение.
2. Команда запуска: npm start
3. Сервер сам отдаёт сайт и WebSocket-комнаты.
4. После деплоя друзья открывают HTTPS-ссылку.
5. В онлайн-комнате сервер обычно подставится сам как wss://адрес-сайта.

Как выпустить игру для всех:
1. Самый простой путь: GitHub Pages, Netlify или Vercel.
2. Загрузи всю папку BlockDrop_Web.
3. Получи публичную HTTPS-ссылку.
4. Отправь ссылку друзьям.
5. На iPhone друзья смогут открыть ссылку в Safari и добавить игру на экран Домой.

Важно:
- GitHub Pages, Netlify и Vercel Static подходят для одиночной игры и челлендж-кодов.
- Для онлайн-комнат нужен Node.js сервер из server.js.
- Сейчас онлайн-режим показывает результаты друзей в комнате в реальном времени.
- В PvP очистка 2+ линий отправляет мусорные линии соперникам.
- В комнате видно призрачную высоту лидирующего соперника на своём поле.
- Турнир завершается по общему таймеру и показывает финальную таблицу.
- После поражения бот-тренер показывает короткий разбор партии.
