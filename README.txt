BlockDrop Web Game
==================

Браузерная игра в духе Tetris с одиночными режимами, тренировкой против ИИ, локальными сохранениями, серверными рекордами, онлайн PvP-комнатами, мобильным управлением, реплеями и PWA-поддержкой.

Демо:
http://45.148.117.119/

Возможности
-----------

- Поле 10x20, генератор фигур 7-bag, SRS wall kicks, ghost piece, hold, очередь следующих фигур, DAS/ARR и lock delay.
- Режимы: Classic, 40 Lines, Hardcore, Time Attack, Zen и Chaos.
- Очки, уровень, линии, таймер, лучший результат, локальная статистика, достижения и серверная таблица лидеров.
- Звуки через Web Audio API, отключение звука, визуальные эффекты и тряска экрана.
- ИИ-соперник с настройкой сложности, стиля и темпа.
- Реплеи и ghost run для лучших локальных попыток.
- Онлайн-комнаты со ссылками `/room/CODE`, QR-приглашением и PvP-атаками через WebSocket.
- Интерфейс на русском и английском.
- Мобильное управление жестами и кнопками.

Управление
----------

- Клавиатура: стрелки или WASD для движения, Up/W/X для вращения, Q для обратного вращения, Space/Z для жёсткого сброса, C/H/E/Shift для hold, P/Esc для паузы.
- Мышь/тачпад: клик для вращения, двойной клик для обратного вращения, перетаскивание для движения и сброса, правый клик для hold.
- Сенсорный экран: тап, двойной тап, свайпы и долгое нажатие для основных действий.

Локальный запуск
----------------

npm install
npm start

Открыть:
http://localhost:8787

`index.html` можно использовать для статической одиночной игры. Онлайн-комнаты и серверные рекорды требуют `server.js`.

Скрипты
-------

npm start
npm run lint
npm test
npm run test:e2e
npm run capture:media
npm run verify

--------------
